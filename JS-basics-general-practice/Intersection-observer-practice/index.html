Intersection Observers!

The Intersection Observer API provides a way to asynchronously observe changes in the intersection of a target element with an ancestor element or with a top-level document's viewport.
In this example the navBar links will change color based on the scroll.

What can it be used for?

Lazy-loading content as a page is scrolled.
Implementing "infinite scrolling" web sites, where more and more content is loaded and rendered as you scroll.
Reporting of visibility of advertisements in order to calculate ad revenues.
And much more.
